# 코드 분석 명령

코드베이스의 종합적인 분석을 수행합니다.

## 수행 작업

### 1. 구조 분석
- 디렉토리 구조
- 모듈 의존성 그래프
- 진입점 및 핵심 컴포넌트

### 2. 품질 분석
- 코드 복잡도 (Cyclomatic Complexity)
- 중복 코드 비율
- 테스트 커버리지
- 린트 이슈

### 3. 의존성 분석
- 외부 의존성 목록
- 버전 현황
- 취약점 검사
- 라이선스 검사

### 4. 보안 분석
- 민감 정보 노출
- 보안 취약점
- 위험 패턴

## 사용법

```
/analyze [경로 또는 옵션]
/analyze                    # 전체 분석
/analyze src/               # 특정 경로
/analyze --type=security    # 보안 분석만
/analyze --type=quality     # 품질 분석만
/analyze --type=deps        # 의존성 분석만
```

## 분석 유형

### 구조 분석 (structure)
- 아키텍처 패턴 식별
- 레이어 분리 확인
- 순환 의존성 탐지

### 품질 분석 (quality)
- 복잡도 메트릭
- 코드 스멜
- 기술 부채 추정

### 의존성 분석 (deps)
- 의존성 트리
- 중복 의존성
- 오래된 패키지

### 보안 분석 (security)
- OWASP Top 10
- 하드코딩된 시크릿
- 취약한 의존성
