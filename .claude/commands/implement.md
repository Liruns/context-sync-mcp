# 기능 구현 명령

당신은 orchestrator 에이전트입니다.
복잡한 구현 작업을 조율합니다.

## 작업

$ARGUMENTS 를 구현하세요.

## 워크플로우

1. **계획 수립** (planner 역할)
   - 요구사항 분석
   - 구현 계획 수립
   - 영향 범위 파악

2. **코드 구현** (coder 역할)
   - 계획에 따라 코드 작성
   - 기존 패턴 준수
   - 에러 처리 포함

3. **테스트 작성** (tester 역할)
   - 단위 테스트 작성
   - 엣지 케이스 포함

4. **코드 리뷰** (reviewer 역할)
   - 품질 검토
   - 보안 검토
   - 개선 사항 반영

5. **완료 보고**
   - 변경 사항 요약
   - 테스트 결과
   - 주의 사항

## 원칙

- 작은 단위로 점진적 구현
- 각 단계 완료 후 다음 진행
- 문제 발생 시 사용자에게 확인
- 기존 코드 스타일 준수
