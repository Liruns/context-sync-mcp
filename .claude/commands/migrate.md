# 마이그레이션 명령

프레임워크, 라이브러리, 언어 버전의 안전한 마이그레이션을 수행합니다.

## 수행 작업

### 1. 현재 상태 평가
- 현재 버전 확인
- 의존성 분석
- 호환성 검사

### 2. 마이그레이션 계획
- 대상 버전 확인
- Breaking changes 분석
- 단계별 계획 수립
- 롤백 계획 준비

### 3. 마이그레이션 실행
- 의존성 업데이트
- 코드 수정
- 설정 파일 업데이트

### 4. 검증
- 테스트 실행
- 기능 확인
- 성능 비교

## 사용법

```
/migrate [대상]
/migrate react@18
/migrate node@20
/migrate typescript@5
/migrate --check  # 마이그레이션 필요 여부 확인
```

## 지원 마이그레이션

### 프레임워크
- React 버전 업그레이드
- Next.js 버전 업그레이드
- Vue 2 → Vue 3
- Angular 버전 업그레이드

### 언어/런타임
- Node.js 버전 업그레이드
- TypeScript 버전 업그레이드
- Python 버전 업그레이드

### 도구
- Webpack → Vite
- Jest → Vitest
- npm → pnpm
