# 성능 최적화 명령

애플리케이션 성능을 분석하고 최적화합니다.

## 수행 작업

### 1. 성능 측정
- 현재 성능 지표 수집
- 병목 지점 식별
- 베이스라인 설정

### 2. 분석
- 핫스팟 식별
- 알고리즘 복잡도 분석
- 리소스 사용 패턴 분석

### 3. 최적화 실행
- 알고리즘 개선
- 캐싱 적용
- 불필요한 연산 제거
- 비동기 처리 최적화

### 4. 검증
- 성능 재측정
- 개선율 계산
- 회귀 테스트

## 사용법

```
/optimize [경로 또는 영역]
/optimize src/api/
/optimize --target=bundle
/optimize --target=database
```

## 최적화 영역

### 프론트엔드
- 번들 사이즈 최적화
- 렌더링 성능
- 이미지 최적화
- 코드 스플리팅

### 백엔드
- API 응답 시간
- 데이터베이스 쿼리
- 메모리 사용
- 동시성 처리

### 알고리즘
- 시간 복잡도 개선
- 공간 복잡도 개선
- 캐싱 전략

## 출력 형식

```markdown
## 성능 최적화 보고서

### 측정 결과
| 지표 | Before | After | 개선 |
|------|--------|-------|------|
| ...  | ...    | ...   | ...  |

### 수행된 최적화
1. [최적화 내용]
2. [최적화 내용]

### 권장 사항
- [추가 최적화 제안]
```

## 관련 에이전트/스킬
- `optimizer` 에이전트 사용
- `performance` 스킬 활용

$ARGUMENTS
